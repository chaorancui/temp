### Linux Shell 命令行：历史命令相关操作
https://learnku.com/server/wikis/36635

```bash
调用命令历史的快捷键：
上箭头键（UP）  ：回溯命令历史。
ctrl + p ： 回溯命令历史。
下箭头键（DOWN）：前溯命令历史。
ctrl + n： 前溯命令历史。
ctrl + r： 进入命令历史查找状态，根据输入字符查找最近执行的历史命令。
ctrl + g： 退出上面的命令历史查找状态。
```
